## About Me

I'm a software engineer at Microsoft working in the site reliability engineering space, where I focus on keeping cloud systems healthy, automated, and resilient for enterprise customers on Azure.

My path into tech is a little unconventional — I came from bartending and found my way into software through [Code Fellows](https://www.codefellows.org), where I learned full-stack JavaScript and fell in love with building things. That people-first background still shapes how I approach engineering: clear communication, empathy, and owning hard problems end-to-end.

- 🔭 Currently building production tooling for reliability engineering at scale
- 🌱 Deep in Python, TypeScript, Azure, and distributed systems
- 💬 Ask me about SRE practices, Azure, production readiness, or career pivots into tech
- ⚡ Fun fact: I used to mix cocktails for a living — turns out debugging prod incidents and a busy bar rush have a lot in common
